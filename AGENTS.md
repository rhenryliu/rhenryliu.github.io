# AGENTS.md

This repository is my personal academic website (R. Henry Liu), built from the
al-folio Jekyll starter and deployed to https://rhenryliu.github.io. It is a
content repository — I edit content here, not the theme runtime (which lives in
external `al_*` gems).

For working guidelines — configuration facts, the local Docker preview loop,
which files are safe to edit, and my review discipline — see **CLAUDE.md**.

Note: this file replaces the upstream al-folio AGENTS.md, which was written for
theme *maintainers* and described a different setup (a `/al-folio` baseurl,
sibling gem repos, CI/style-contract tooling). None of that applies to this site.
